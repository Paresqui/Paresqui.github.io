INTELIGÊNCIA DE NEGÓCIOS 
_____

Business Intelligence (BI) ou Inteligência de Negócios, é um termo utilizado para descrever uma série de conceitos que permitem coletar, transformar, gerenciar e distribuir os dados para descobrir insights valiosos.

O BI consiste em um conjunto de técnicas utilizadas para extrair inteligência a partir de dados sobre um determinado negócio.

Que possui como objetivo principal transformar dados brutos em informações compreensíveis e significativas que orientam a tomada de decisão.

A QUEM SE DESTINA?
_____

A todas as organizações que desejam fazer melhor uso dos deus dados para retirar informações relevantes que suportem a tomada de Decisão.

Isso se faz necessário, pois num cenário cada vez mais competitivo, não é mais possível tomar decisões baseadas apenas no feeling.

COMO FUNCIONA?

ETL - Extract, Transform, Load

Obtenção, centralização, limpeza, tratamento e criação do modelo de dados.

Elaboração de Relatórios

Desenvolvimento de relatórios.
