DASHBOARD DE VENDAS B2C

Dashboard de Vendas construído na plataforma Microsoft Power BI e desenvolvido para a empresa Raw Fitness. A empresa e a base de dados utilizada nesse projeto são fictícias e possuem o intuito de mostrar como as visualizações e todo o processo de Análise de Dados funcionam na prática.

O painel conta com análise de vendas, descontos, lucro e detalhes de vendas por segmento, produto, ano e país. Para exibir o relatório completo, clique aqui.
