DASHBOARD HISTÓRICO DE COTAÇÃO DO DÓLAR

Dashboard contendo histórico da cotação do dólar ao longo do tempo, construído na plataforma Microsoft Power BI. Os dados foram obtidos através da API disponibilizada pelo Banco Central e a base apresenta dados desde o ano 2000.

O painel conta com análise da cotação do dólar ao longo do tempo, cotação média anual e cotação média mensal para o ano de 2021. Para exibir o relatório completo, clique aqui.
