DASHBOARD DE RECURSOS HUMANOS COM ANÁLISE DE TURNOVER 

Dashboard de Recursos Humanos construído na plataforma Microsoft Power BI e desenvolvido para a empresa GBO Human Resources. A empresa e a base de dados utilizada nesse projeto são fictícias e possuem o intuito de mostrar como as visualizações e todo o processo de Análise de Dados funcionam na prática.

O painel conta com análises de contratações, demissões, cálculo de turnover e análise de perfil do colaborador, para exibir o relatório completo, clique aqui.
